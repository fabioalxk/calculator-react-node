# Project Setup

Open two terminals and run the following commands:

---

### Terminal 1️⃣

```sh
$ cd calculator-api
$ npm i
$ npm start
```

---

### Terminal 2️⃣

```sh
$ cd calculator-react-app
$ npm i
$ npm start
```

---

### How to run the tests ✅

```sh
$ mocha
```

The application will start on [http://localhost:3000]
